# MyPortofolio

Welcome to the **MyPortofolio** repository! This is my personal portfolio website, built to showcase my skills, projects, certifications, and professional journey as a Full Stack Developer from Amman, Jordan.

## Live Demo

Check out the live portfolio here: [MyPortofolio Live](https://jacklusy.github.io/MyPortofolio/)

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript, Tailwind CSS
- **Backend:** PHP, Laravel
- **Tools:** Git, GitHub

## Features

- **Responsive Design:** Optimized for all devices.
- **Project Showcase:** Detailed project pages with live demos and source code links.
- **Professional Profile:** Includes work history, certifications, skills, and contact information.
